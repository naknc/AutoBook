# AutoBook User Guide

## Search

- Use `Catalog Search` to search public-domain and external sources.
- Apply source, language, format and rating filters after results load.
- Use `AI Suggestions` if an API key is configured and you want alternative queries.

## Library

- Use collections, favorites, tags and reading status to organize titles.
- Turn on bulk mode for batch collection, OCR or conversion actions.
- Use repair, OCR, conversion and AI enrichment actions directly from each book card.

## History and Queue

- `Download History` tracks successful and failed downloads.
- The queue section supports run, retry, cancel and reorder actions.

## Devices

- Configure device profiles in Settings.
- If a profile has `auto_send` enabled, matching devices can receive books automatically after download.

## Settings

- Control allowed sources, formats and workspace actions.
- Manage offline cache, plugin manifests and companion/web preview outputs.
- Health scan and organize tools are available from the actions row.
